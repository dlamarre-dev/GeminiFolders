Google Gemini를 매일 사용하고 있나요 — 업무, 코딩, 글쓰기, 연구에 — 하지만 같은 프롬프트를 반복해서 입력하고, 소중한 대화들은 끝없는 기록 속으로 사라지나요?
Gemini 폴더 (Gemini Folders) 가 바로 당신에게 필요한 확장 프로그램입니다.

버전 4.0으로 Gemini 폴더는 진정한 Gemini 컨트롤 센터가 됩니다. 대화를 폴더로 정리하고, 재사용 가능한 프롬프트 라이브러리를 구축하고, 복사 붙여넣기 없이 클릭 한 번으로 Gemini에 직접 주입하세요. 중요한 프롬프트를 라이브러리 상단에 고정.

✨ v4.0 신기능 — 프롬프트 모드

📝 개인 프롬프트 라이브러리: 헤더의 토글 버튼을 한 번 클릭해 프롬프트 모드로 전환하세요. 제목과 전체 텍스트로 최고의 프롬프트를 저장하고, 검색 및 정렬(최신순, 오래된순, A-Z)로 즉시 찾을 수 있습니다.
▶ 원클릭 주입: 복사 붙여넣기는 이제 그만. 저장된 프롬프트의 재생 버튼 (▶)을 클릭하면 Gemini 입력창에 직접 주입됩니다. Gemini 페이지에 있는지 확인하고 그렇지 않으면 알림을 표시합니다.
✏️ 인라인 편집: 프롬프트를 클릭해 펼친 후 텍스트를 바로 편집하세요. 변경 사항은 짧은 시간 후 자동 저장되고 정렬 순서도 업데이트됩니다.
💎 커스텀 Gem 링크: 즐겨찾는 Gem 링크를 설정하고 클릭 한 번으로 활성화하세요. 활성화 시 "새 대화" 버튼이 기본 채팅 대신 내 Gem을 엽니다.
☁️ 프롬프트 동기화: 폴더와 독립적으로 프롬프트 라이브러리의 클라우드 동기화를 활성화하세요.
📌 고정 프롬프트: 중요한 프롬프트를 클릭 한 번으로 라이브러리 상단에 고정합니다. 시각적 구분선으로 나머지와 구분됩니다.

🌟 폴더 기능

⚡ 빠른 저장 (우클릭 & 단축키): 간단한 우클릭으로 대화를 저장하거나, 키보드 단축키(Ctrl+Shift+S 또는 Mac의 Cmd+Shift+S)로 즉시 "⚡ 빠른 저장"에 보냅니다 — 이미 저장된 경우 시각적 알림 표시!
☑️ 다중 선택 (일괄 작업): 체크박스로 여러 대화를 선택해 한 번에 이동하거나 삭제합니다.
📑 탭 그룹 통합: 폴더 전체를 클릭 한 번으로 기본 Chrome 색상 탭 그룹으로 엽니다.
😃 커스텀 이모지: 폴더 이름을 이모지로 시작하면(예: "💻 코드") 기본 아이콘이 교체됩니다.
🖱️ 드래그 앤 드롭: 폴더 간에 대화를 드래그해 작업 공간을 직관적으로 재구성하세요.
📱 모바일 동기화: 토글 하나로 Chrome 북마크에 미러 폴더가 생성되어 폴더 구조와 정렬 순서가 스마트폰에 완벽히 반영됩니다.
🗜️ 초고효율 압축: 데이터를 자동 압축(LZString)해 Chrome 기본 저장 공간을 최대화하고 수백 개의 대화를 저장합니다.
📊 저장 공간 추적기: 시각적 진행 표시줄로 클라우드 저장 공간을 실시간 확인합니다.
📁 커스텀 폴더 & 고정: 프로젝트, 주제, 카테고리별로 대화를 분류하세요. 즐겨찾는 폴더를 (📌) 상단에 고정합니다.
⇅ 커스텀 정렬: 폴더와 대화를 즉석에서 정렬하고(최신순, 오래된순, A-Z) 모바일 동기화에 즉시 반영됩니다.
🤖 스마트 감지 & 검색: 대화 제목 자동 입력 및 실시간 검색창.
☁️ 클라우드 동기화 & 백업: 모든 기기에서 정리 상태에 접근하고 JSON으로 데이터를 내보냅니다.
🎨 Material 3 디자인 & 다국어: 완전 커스텀 모달 다이얼로그, 자동 다크 모드, 16개 언어 지원의 초소형 인터페이스!

🛠️ 어떻게 사용하나요?

폴더 모드:
gemini.google.com에서 대화를 엽니다.
빠른 저장: Ctrl+Shift+S(Mac은 Cmd+Shift+S), 또는 우클릭 > "Gemini 폴더에 저장".
확장 프로그램으로: 아이콘 클릭 > 폴더 선택 > 저장.
모바일: 하단의 📱 를 활성화해 폴더를 스마트폰과 동기화하세요.
체크박스로 일괄 작업, 탭 그룹(📑)으로 폴더 열기, 드래그 앤 드롭으로 재구성!

프롬프트 모드:
확장 프로그램 오른쪽 상단 토글의 📝 를 클릭합니다.
➕ 프롬프트 추가를 클릭하고 제목과 텍스트를 입력 후 저장합니다.
프롬프트 사용: 펼친 후 ▶ 클릭 — Gemini에 직접 주입되어 바로 전송 가능합니다.
💎 를 설정해 커스텀 Gem을 지정하세요.
프롬프트 고정: 📍 클릭 — 목록 상단으로 이동하여 항상 바로 접근할 수 있습니다.

🔒 개인정보 보호 & 보안
개인정보 보호를 핵심으로 설계되었습니다. gemini.google.com에만 접근 권한을 요청하며, 대화 저장 또는 프롬프트 주입 시에만 활성 탭을 읽습니다. "북마크" 권한은 모바일 동기화 활성화 시에만 사용됩니다. Chrome Sync로 저장: 서드파티 서버 없음, 추적 없음.

🤝 오픈소스 프로젝트
Gemini 폴더는 열정으로 개발된 오픈소스 프로젝트입니다. 투명성과 개인정보 존중이 최우선입니다. 개발자이거나 단순히 궁금하신가요? 코드 확인, 변경 이력 추적, 또는 GitHub에서 기여하세요: https://github.com/dlamarre-dev/GeminiFolders
