Οι συνομιλίες AI σας είναι διασκορπισμένες σε πέντε διαφορετικά sites. Συνεχίζετε να επαναπληκτρολογείτε τα ίδια prompts. Οι καλύτερες συνομιλίες σας εξαφανίζονται σε ατελείωτες ιστορικές καταγραφές στο Gemini, Claude, ChatGPT, Copilot και Perplexity.

Φάκελοι ΤΝ είναι το ενοποιημένο AI cockpit σας: οργανώστε συνομιλίες από όλα τα AI εργαλεία σας σε κοινόχρηστους φακέλους, δημιουργήστε μια βιβλιοθήκη επαναχρησιμοποιήσιμων prompts και εισάγετε prompts απευθείας σε οποιοδήποτε υποστηριζόμενο AI — χωρίς αντιγραφή και επικόλληση.

✨ ΛΕΙΤΟΥΡΓΙΑ PROMPT

📝 Προσωπική βιβλιοθήκη prompt: Αλλάξτε στη Λειτουργία prompt με ένα κλικ στον διακόπτη στην κεφαλίδα. Αποθηκεύστε τα καλύτερα prompt σας με τίτλο και κείμενο, βρείτε τα άμεσα με αναζήτηση και ταξινόμηση.
▶ Εισαγωγή με ένα κλικ: Κάντε κλικ στο κουμπί αναπαραγωγής (▶) σε οποιοδήποτε αποθηκευμένο prompt για να το εισάγετε απευθείας στο πεδίο εισόδου του ενεργού AI.
✏️ Inline επεξεργασία: Κάντε κλικ για να αναπτύξετε ένα prompt και επεξεργαστείτε το κείμενο απευθείας. Οι αλλαγές αποθηκεύονται αυτόματα.
🔀 Κουμπιά Νέας Συνομιλίας: Ξεκινήστε μια νέα συνομιλία στο ChatGPT, Claude, Copilot, Gemini ή Perplexity απευθείας από τη Λειτουργία Prompt — ένα κουμπί ανά υπηρεσία.
☁️ Συγχρονισμός prompt: Ενεργοποιήστε συγχρονισμό cloud για τη βιβλιοθήκη prompt ανεξάρτητα από τους φακέλους.
📌 Καρφιτσωμένα prompt: Καρφιτσώστε σημαντικά prompt στην κορυφή της βιβλιοθήκης.
⌨️ Εκκίνηση prompt με ζωντανές προτάσεις: Πληκτρολογήστε # στο πεδίο συνομιλίας οποιασδήποτε AI για να δείτε αμέσως όλα τα αποθηκευμένα prompt σας — φιλτραρισμένα σε πραγματικό χρόνο καθώς πληκτρολογείτε. Πατήστε Διάστημα σε μοναδική αντιστοίχιση για αυτόματη συμπλήρωση ή σε ακριβή αντιστοίχιση για άμεση εισαγωγή. Λειτουργεί με ChatGPT, Claude, Gemini, Copilot και τοπικά LLM.

🏠 ΥΠΟΣΤΗΡΙΞΗ ΤΟΠΙΚΟΥ LLM

Χρησιμοποιείτε τοπικό μοντέλο AI (Ollama, LM Studio, Jan ή άλλο); Ρυθμίστε τα Φάκελοι ΤΝ με την τοπική σας URL — υποστηρίζει localhost και διευθύνσεις LAN (192.168.x.x, προσαρμοσμένα ονόματα κεντρικού υπολογιστή). Η γρήγορη αποθήκευση, η αποθήκευση μέσω μενού περιβάλλοντος και η έγχυση προτροπών λειτουργούν στις περισσότερες τοπικές διεπαφές (συμπεριλαμβανομένου του Open WebUI).

🛠️ ΠΩΣ ΛΕΙΤΟΥΡΓΕΙ;

Λειτουργία φακέλων:
Ανοίξτε μια συνομιλία στο Gemini, Claude, ChatGPT, Copilot, Perplexity.
Άνοιγμα πλαισίου: Ctrl+Shift+1 (ή Cmd+Shift+1 σε Mac).
Γρήγορη αποθήκευση: Ctrl+Shift+S (ή Cmd+Shift+S σε Mac), ή δεξί κλικ > «Αποθήκευση στο Φάκελοι ΤΝ».
Μέσω της επέκτασης: Κάντε κλικ στο εικονίδιο, επιλέξτε φάκελο και αποθηκεύστε.
Κινητό: Επιλέξτε 📱 στο υποσέλιδο για συγχρονισμό φακέλων με το κινητό σας.

Λειτουργία prompt:
Κάντε κλικ στο 📝 στον διακόπτη επάνω δεξιά της επέκτασης.
Κάντε κλικ στο ➕ Προσθήκη prompt, εισάγετε τίτλο και κείμενο και αποθηκεύστε.
Χρήση prompt: αναπτύξτε το και κάντε κλικ ▶ – εισάγεται απευθείας στον ενεργό AI.
Καρφίτσωμα prompt: κάντε κλικ 📍 – μετακινείται στην κορυφή της λίστας.
Εκκίνηση prompt: Πληκτρολογήστε # για να δείτε αμέσως όλα τα αποθηκευμένα prompt σας, ή συνεχίστε να πληκτρολογείτε για φιλτράρισμα σε πραγματικό χρόνο. Πατήστε Διάστημα σε μοναδική αντιστοίχιση για αυτόματη συμπλήρωση ή σε ακριβή αντιστοίχιση για εισαγωγή.
